# apple-ar-quick-look-demo

Native iOS AR prototype demonstrating AR Quick Look for placing and viewing 3D objects in real-world environments using ARKit.

---

## 🧠 System Overview

- Loads USDZ 3D assets
- Displays them in AR using AR Quick Look
- Anchors objects in real-world space via ARKit

---

## 🔌 Key Concepts Demonstrated

- AR Quick Look integration
- USDZ asset pipeline
- Real-world object placement
- Lightweight AR interaction flows

---

## 🛠️ Tech Stack

- Swift
- ARKit
- AR Quick Look

---

## ⚙️ Setup

1. Clone the repo
2. Open in Xcode
3. Run on ARKit-supported iOS device

---

## 📌 Notes

- Focused on Apple ecosystem
- Minimal interaction layer (viewer-style experience)
